# Exports

## TreeFactory preset

Editable lab state that can be imported back into TreeFactory.

## Nexus object JSON

Contains:

```txt
nexus-object-descriptor/1
procedural object material descriptor
procedural object LOD descriptor
procedural object capture profile
capture plan
tree asset snapshot
```

## GLB

Exports the current LOD0 Three.js object.

## Impostor PNG

Exports the atlas generated by the current Three.js tree adapter.
